# Температурный контроллер на Arduino (DS18B20 + Реле)

## Описание

Проект представляет собой простой температурный контроллер на базе Arduino.

Система измеряет температуру с помощью цифрового датчика **DS18B20** и автоматически включает реле при превышении заданного порога.

- Температура считывается каждую секунду.
- Если температура выше **30°C**, реле включается.
- Если температура **30°C или ниже**, реле выключается.
- Текущая температура и состояние реле отображаются в Serial Monitor.

---

## Используемые компоненты

- Arduino Uno/Nano (или совместимая плата)
- Датчик температуры DS18B20
- Реле 5 В
- Резистор 4.7 кОм (между DATA и VCC датчика)
- Соединительные провода

---

## Подключение

| Устройство | Arduino |
|------------|----------|
| DS18B20 DATA | D2 |
| DS18B20 VCC | 5V |
| DS18B20 GND | GND |
| Реле IN | D3 |
| Реле VCC | 5V |
| Реле GND | GND |

Для DS18B20 обязательно подключите подтягивающий резистор **4.7 кОм** между выводами **DATA** и **VCC**.

**Sketch → Include Library → Manage Libraries**

---

## Логика работы

1. Инициализация датчика температуры.
2. Каждую секунду производится измерение температуры.
3. Если температура превышает **30°C**, включается реле.
4. Если температура ниже либо равна **30°C**, реле отключается.
5. Информация выводится в Serial Monitor.

---

## Пины

```cpp
DS18B20 -> D2
Relay   -> D3
```

---

## Порог включения

В коде используется следующий порог:

```cpp
if (temperature > 30.0)
```

При необходимости можно изменить значение:

```cpp
if (temperature > 25.0)
```

или

```cpp
if (temperature > 40.0)
```
